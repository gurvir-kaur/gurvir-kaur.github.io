What does the term Internet means ?

The Internet, sometimes called simply "the Net" is a worldwide system of computer networks - a network of networks in which users at any one computer can, if they have permission, get information from any other computer (and sometimes talk directly to users at other computers). The Computers are interconnected by the use of the Internet protocol suite (TCP/IP) which links the devices worldwidely.


LAN, (local area network)

Ethernet segments - two parts, one is Ethernet wire and second is a hub, hub has ports into which the different hosts (computers) can be connected through RJ45(Registered Jack) modular plugs. Hub is a device that copies the data on all its ports, which it receives through any one of the ports, all the computers connected through hub can see all the data.
If we want to connect multiple hubs together we need a device called Bridge. Multiple Ethernet bridges are connected via bridges. Bridges don't broadcast the data but forward data from one hub to another only when it needs to get there.

Different LANs speak different languages so how do different LANs get to communicate with each other? ROUTERS! Routers are the devices that help to fulfil this task. When we bring routers into the picture and start connecting different LANs together, we get a WAN (Wide Area Network).

The most common WAN available is called the Internet, but internet is not only the one WAN available out there. There is also an another type of WAN available out there called MAN (Metropolitan Area Network)- network that can span across a city and help in city networking and communication.

Two terms, internet and Internet, the former is more generic which means just an interconnection of different type of networks such as LANs, MANs and WANs. The latter is the most popular internet, the Internet.
The core of Internet is TCP/IP (Transmission Control Protocol/Internet Protocol), these are language or communication standards, in order for any computer to be on Internet it has to be able to speak or understand TCP/IP.
Now, there are two jobs of TCP/IP :

  1. Well defined behaviour of how TCP/IP sends data from one computer to the other, data sends in the form of packets or datagrams.
  2. It is able to uniquely identify any computer on the Internet; the way this is done via the special number IP address.
But, we don't use IP addresses to search on Internet, we use Domain Names, that is a human readable text that translates into IP address and on the right side of a domain name we have .com or .edu or .gov, which tells us about the first levels of domain names and on the left of them we have second level domain names such as google or amazon now the second level domain names can further have third level foe examplemaps.google.com

Now how do we know that maps.google.com will find a special IP address somewhere to hit the google map server?
So, there is a huge database worldwide called DNS which maps domain names to the IP addresses, until 1988 these domain names were kept in one single text file, but now there might be a whole database somewhere or cloud
The biggest power of Internet is that we can send or receive data literally from anyone who is connected to the Internet. We get through to the Internet via ISP (Internet Service Providers)

Two things to know about them-
  1. They give us access to Internet.
  2. They tell us to restart our modems when Internet breaks.
So, to conclude there are two things that make up the Internet the wires or optical fibres creating those WANs and TCP/IP protocols.
